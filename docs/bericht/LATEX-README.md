### Wie ist dieser Unterordner zu nutzen?

- main.tex enthält den allgemeinen Header, sowie eine Vorlage
- jeder bekommt einen eigenen Unterordner, der genau diese Vorlage schon enthält &rarr; in diesen Unterordnern arbeitet jeder an seinem eigenen Text
- bitte NICHTS in main.tex verändern
- bitte keine Veränderungen am Header, ohne es mir explizit zu sagen oder es eindeutig im eigenen File zu markieren
- wenn neue Pakete verwendet werden &rarr; entweder Bescheid geben oder eindeutig kennzeichnen
- auch Bilder, bibtex, etc. im eigenen Unterordner ablegen, wenn gebraucht
- am Ende füge ich die Unterordner zusammen
